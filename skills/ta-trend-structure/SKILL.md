# 趋势结构分析

> **运行时只读本文件。** `modules/skill_loader.load_skill("ta-trend-structure")`

## 何时使用

- 趋势方向、支撑阻力、市场状态（牛/熊/震荡/弱反弹/恐慌）
- 大盘环境约束：道氏双指数验证、市场广度、日/周/月组合
- 为其他分析提供**环境约束**（权重加减）

## 输入数据

**个股**：`quotes` 的 `high`, `low`, `close`, `volume`；`indicators.ma20`, `ma60`。

**环境（可选，`python scripts/sync_market.py`）**：
- 指数：`INDEX.SH000001`（上证）、`INDEX.SZ399001`（深证）、`INDEX.SH000300`（沪深300）
- `market_breadth` 表：上涨/下跌家数、涨跌停

`modules/trend_structure.analyze_trend_structure` 输出 `trend`, `regime`, `support`, `resistance`。

## 核心判据

### 峰谷与趋势（道氏定义）

| 迹象 | 含义 |
|------|------|
| 高点更高 + 低点更高 | 上升趋势 |
| 高点更低 + 低点更低 | 下降趋势 |
| 高低点均未突破前极值 | 横盘/震荡（约 1/3 时间处于「无趋势」） |
| 上升中未能越过前高 | 趋势转弱第一警告 |
| 下降中未能跌破前低 | 空侧转弱警告 |
| 支撑有效击穿（收盘确认） | 上升结构破坏 |
| 阻力有效突破（收盘确认） | 下降结构破坏 |
| 原支撑跌破后反弹受阻 | 支撑→阻力（极性转换） |

趋势分三种**规模**：主要（A 股日线可近似为月级）、次要（周级）、短暂（日级）；分析时以**长期定方向、短期定时机**。

### 道氏双指数相互验证（A 股映射）

原书：道琼斯工业 + 运输指数须**同向确认**。A 股映射：**上证指数 + 深证成指**（或沪深300 + 中证500）。

| 组合 | 解读 |
|------|------|
| 两指数收盘均突破各自前一轮中趋势峰值 | 牛市/上升趋势确认信号 |
| 两指数收盘均低于各自前一轮反弹谷 | 熊市/下降趋势确认信号 |
| 仅一指数创新高/新低，另一未确认 | **不能**认定大趋势反转；原趋势仍有效 |
| 两指数方向背离（一创新高、一未跟） | 警告：相互背离，趋势可信度降权 |
| 个股突破 + 双指数未同向确认 | 标注逆势，降置信度 |

买卖信号以**收盘价**穿越前高/前低为准；日内刺穿无效。

### 趋势线（第三点验证）

| 迹象 | 含义 |
|------|------|
| 上升：连接两个依次抬高的反弹低点 → 试画趋势线 | 尚为试验性 |
| 价格第三次触及趋势线并再次反弹 | 趋势线**有效** |
| 价格有效跌破上升趋势线（收盘） | 上升趋势反转信号 |
| 价格有效突破下降趋势线（收盘） | 下降趋势反转信号 |
| 被突破的旧趋势线 | 常转为反向支撑/阻力 |
| 仅两点即画线并交易 | 误判：缺第三点验证 |

### 管道线（返回线，次于趋势线）

**优先级**：基本趋势线 > 管道线（page-0100）；管道为辅助工具。

| 迹象 | 含义 |
|------|------|
| 上升趋势：连接两个抬升低点画趋势线，再自第一个显著峰引平行线 | 试画管道 |
| 下一轮上涨触管道线后折返，且跌回趋势线 | 管道**一半**成立 |
| 折返再次在趋势线获支撑 | 管道**基本确认** |
| 价格有效突破管道上/下轨（收盘） | 趋势加速或反转预警 |
| 管道未被触及越久、试探成功次数越多 | 管道越可靠 |

下降管道镜像：连接两个降低高点为趋势线，自第一个显著谷引平行线。

### 百分比回撤（区间读法）

除单点 50%/33%/66% 外，可将回撤区合并理解（page-0100）：

| 回撤区 | 范围 | 含义 |
|--------|------|------|
| 最小回撤区 | **33%～38%** | 强趋势常见浅调 |
| 中等回撤区 | **38%～50%** | 多数调整在此获支撑 |
| 最大回撤区 | **62%～66%** | 超过则反转概率 > 普通调整 |
| 折中参考 | 40% / 60% | 部分分析者采用的简化区 |

单点判据（与上表并用）：

| 回撤位 | 含义 |
|--------|------|
| **50%** | 最常见回撤深度；道氏次要趋势常见约一半 |
| **33%（1/3）** | 强趋势中常见**最小**回撤 |
| **66%（2/3）** | 强趋势中常见**最大**回撤 |
| **38% / 62%** | 斐波那契回撤；多数调整在 38%～50% 间获支撑 |
| 上升回调至 50%～62% 且缩量企稳 | 趋势内健康调整，偏多 |
| 回撤超过 66% 或跌破前谷 | 结构转弱，不宜视为普通回调 |

### 正常反弹 vs 双重顶/底（勿混淆，page-0133）

| 情形 | 处理 |
|------|------|
| 上升中价格从前高正常挡下 | **正常**，不是双重顶 |
| 下降中价格从前低正常反弹 | **正常**，不是双重底 |
| 仅当**收市价跌破**两顶之间的低点（颈线/中间低） | 双重顶才成立 |
| 仅当**收市价突破**两底之间的高点 | 双重底才成立 |
| 形态时间尺度 | 常 2～3 个月；周线/月线可跨年 |

### 日 / 周 / 月多周期组合

| 组合 | 含义 | 策略 |
|------|------|------|
| 月、周、日同向 | 突破态，趋势良好 | 顺势交易，信号加权 |
| 三周期不一致 | 平衡态，趋势不良 | 区间交易或观望；**以长期方向为最终依据** |
| 日线转弱但周线仍强 | 次要调整 | 不宜直接定熊 |
| 日线突破前高/低 | 短期转变 | 继续观察周线是否确认 |

**K 线中点辅助**（单根完成后的峰谷分析）：上升结构中，若第三根 K 维持于第二根中点之上 → 强势；落在中点与低点之间 → 正常调整；跌破第二根低点 → 方向转弱。下降结构镜像处理。

### 市场广度

指数只反映样本股；广度揭示「大军是否跟上将军」。

| 迹象 | 含义 |
|------|------|
| 指数上涨 + 上涨家数 > 下跌家数 + AD 线同向 | 广度健康，趋势可信 |
| 指数创新高 + AD 线（涨跌家数累积线）未创新高 | AD 背离，顶背离警告 |
| 指数收高但下跌家数 > 上涨家数 | 指数失真，广度负面，降权做多 |
| 52 周新高家数 > 新低家数 | 环境偏多 |
| 指数弱 + 广度强（少数指数失真） | 个股机会与指数背离，注明环境 |

数据缺失时：标注「广度未验证」，不因单一指数结论加权。

**本地数据局限（真实使用必遵）**：

| 数据 | 状态 | Agent 处理 |
|------|------|------------|
| `market_breadth` 涨跌家数 | sync_market 可得 | 可用，但**不能**等同 AD 线 |
| **AD 累积线** | 暂无序列 | 标注「AD 未验证」，不得捏造 AD 背离 |
| **52 周新高/新低家数** | 暂无 | 标注未验证 |
| **周线/月线** | 由日线 resample 近似 | 月线结论**降权**，须写明近似 |
| **个股相对强度** | `analyze_relative_strength` vs 沪深300 | 有指数数据时可用 |

### 个股相对强度（第 18 章）

| 迹象 | 含义 |
|------|------|
| 个股近 20～60 日涨幅显著 > 基准指数 | 相对强势，突破可信度↑（仍看大盘环境） |
| 个股弱、指数强 | 逆势，做多降权 |
| 个股强、指数弱 | 注明「与指数背离的独立行情」 |
| 无基准指数数据 | 标注「相对强度未验证」 |

### 辅助水平线 / 百分比水平（第 2 章，page-0054）

道氏辅助工具：在前高/前低或整数关口画**水平线**；与支撑阻力区、回撤位并用，非精确点。

| 迹象 | 含义 |
|------|------|
| 多次测试同一水平 | 支撑/阻力加强 |
| 收盘突破水平线 | 同趋势线规则，须确认 |
| 仅日内刺穿 | 无效 |

### 多指数相对比较（A 股扩展，page-0405）

除上证+深证道氏验证外，可对照**不同样本指数**强弱：

| 迹象 | 含义 |
|------|------|
| 大盘指数突破 + 更广角指数（如全 A/中证全指）同向突破 | 信号更持久 |
| 大市值指数强、小盘指数弱 | 结构性行情，个股分化 |
| 主要指数与成长/科技指数背离 | 标注风格背离，降权单一指数结论 |

本地 MVP 若无多指数 DB：人工标注或后续 index sync。

### 市场状态 regime

| 状态 | 含义 | 对其它分析 |
|------|------|------------|
| bull | 牛市 | 趋势/突破信号加权；RSI 超买可钝化 |
| bear | 熊市 | 做多降权；超卖勿盲目抄底 |
| range | 震荡 | 重区间边界，轻趋势；MACD 交叉频繁 |
| weak_rebound | 弱反弹 | 碰 MA20/前高受阻即偏空 |
| panic | 恐慌 | 暂停追势；等缩量止跌 2～3 日 |

### 环境 × 信号约束

| 环境 | 个股信号 | 处理 |
|------|----------|------|
| range | MACD 金叉 | 仅区间下沿有效 |
| bear | KDJ 低位金叉 | 短线反弹，非反转 |
| bull | RSI>70 | 不机械看空 |
| panic | 任何做多信号 | 全部降级 |
| 双指数背离 | 突破/趋势信号 | 降置信度 |
| 广度 AD 背离 | 指数看多结论 | 降权或观望 |
| 指数弱 + 个股强 | 突破 | 标注逆势，降置信度 |

### 常见误判

- 单根 K 或日内刺穿即定反转（需**收盘** + 峰谷第三极值）
- 忽略趋势级别（日线 vs 周线 vs 月线）
- 支撑阻力当精确点（应为区域）
- 趋势线两点即画（需第三点验证）
- 只看单一指数、不看双指数验证与广度
- **把触前高回落当成双重顶**（须破中间低才确认）
- 把 38%～50% 正常回撤当成趋势终结

### 失效条件

- K 线 < 20 根
- 除权、复牌、一字板连续
- 重大消息覆盖技术位
- 缺少指数/广度数据却做大盘 regime 强结论

## 分析步骤

1. 峰谷分析定 `trend`（必要时用 K 线中点辅助）
2. 近 20 日高低定支撑/阻力；标注 **33～38% / 50% / 62～66%** 回撤区
3. 若有双指数：检查相互验证或背离
4. 若有广度数据：对照 AD/涨跌家数/新高新低
5. 日/周/月组合定环境（长期优先）
6. 结合 MA、涨跌幅、量比定 `regime`
7. 输出趋势 + 状态 + **对其它 Skill 的权重建议**

## 禁止事项

- K 线不足判长周期
- 无双指数/广度数据却给出强大盘确认
- 忽略指数环境（待指数 sync 扩展后仍须标注数据来源）

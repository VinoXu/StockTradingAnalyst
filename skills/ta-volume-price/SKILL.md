# 量价分析

> **运行时只读本文件。** `modules/skill_loader.load_skill("ta-volume-price")`

## 何时使用

- 放量突破、缩量回踩、放量滞涨、缩量阴跌
- 胀爆（顶部）/ 抛售高潮（底部）
- 价平量增（Churning）、OBV 式价量背离
- 任何技术结论需成交量确认

## 输入数据

`quotes`：`volume`, `turnover_rate`, `amount`, `close`；`indicators.volume_ratio`（当日量/近5日均量）。

**地位**：价格最重要；成交量次之，主要作**验证**；持仓量仅期货（A 股用主力净流入等作辅助，非官方持仓）。

## 核心判据

### 基本配合（趋势段内）

| 迹象 | 含义 |
|------|------|
| 价涨 + 量比>1.2 | 价涨量增，验证上升 |
| 价涨 + 量比<0.8 | 价涨量缩，动能待确认 |
| 价跌 + 量比>1.2 | 价跌量增，抛压大 |
| 价跌 + 量比<0.8 | 价跌量缩，或调整末端或阴跌无力 |
| 量比>1.5 | 明显放量，**看位置与阶段**定多空 |
| 量比<0.7 | 明显缩量；上升回踩常健康，下跌常无力 |
| 换手率异常高 | 分歧加大 |

**趋势内细则**：
- 上升趋势：上涨日量应较重，回调日量应较轻 → 验证趋势；若创新高而量反而萎缩 → **量价背离**，买压减弱预警。
- 下降趋势：下跌日量应较重，反弹日量应较轻 → 验证下跌；若下跌中量持续放大 → 关注底部信号酝酿。

### 位置 × 阶段 × 量能（同一放量，含义不同）

| 阶段/位置 | 放量特征 | 解读 |
|-----------|----------|------|
| **突破关键位**（形态/阻力） | 放量 | 真突破概率↑；向上突破量比向下更重要 |
| **突破关键位** | 缩量 | 假突破概率↑ |
| **上升回调至支撑** | 缩量 | 健康回踩 |
| **趋势中段** | 沿趋势方向放量 | 趋势延续验证 |
| **趋势末段/高位平台** | 放量滞涨（价平或微涨） | 派发预警 |
| **主要顶部** | **胀爆**：长期上涨后价急冲 + 巨量 | 负面警告；期货常伴持仓下降（A 股看主力是否同步流出） |
| **主要底部** | **抛售高潮**：长期下跌后价急跌 + 巨量 | 恐慌出清，常随后急剧反弹；期货常伴持仓大降 |
| **低位** | 放量阳线 | 可能有资金介入 |
| **下跌末段** | 放量长下影 | 可能有恐慌出清 |
| **横盘/窄区间** | **价平量增**（Churning）：多日振幅极小而量显著高于均量 | 筹码换手激烈、方向未定；高位多偏派发，突破前多属蓄势（须等方向选择） |

### 形态完成时的量能（与 `ta-price-patterns` 联动）

| 形态 | 量能要点 |
|------|----------|
| 头肩顶 | 左肩量重；头上冲量轻；跌破颈线初段量急增非必须，但反扑应量轻、再跌应量重 |
| 双重顶/三重顶 | 后继峰量逐次减轻；跌破颈线应放量 |
| 三角/旗形等持续形态 | 形成过程中量萎缩；突破时必须明显放量；反扑量轻 |
| 所有形态完结 | 有效突破应伴随较重交易活动 |

### 涨日粗/跌日细（粗细柱状图，page-0405）

| 迹象 | 含义 |
|------|------|
| 上升趋势 + 上涨日量重、回调日量轻 | 趋势验证（与价涨量增一致） |
| 下降趋势 + 下跌日量重、反弹日量轻 | 下跌验证 |
| 上涨日量反而持续萎缩 | 量价背离，买压减弱 |
| 粗细蜡烛/柱状图 | 量决定实体粗细；与 OBV 方向判读可互证 |

### OBV / 权衡成交量（逻辑，本地可选）

- 涨日量计入 OBV 正、跌日负，看 OBV **方向**是否与价格趋势一致。
- OBV 与价格趋势不协调 → 相互背离，趋势或反转预警。
- 本地若无 OBV 字段：用近 5～10 日「涨日量合计 vs 跌日量合计」或 `main_net_inflow` **近似**，须标注「非标准 OBV」。

### 成交量领先于价格

- 量变化可判断买卖力量消长，**领先**于价格实际反转。
- 量增 + 趋势方向一致 → 趋势或延续；量与持仓（期货）双降 → 趋势或终结预警。

### 组合

| 组合 | 解读 |
|------|------|
| 突破 + 放量 + 主力净流入 | 三向配合，偏多 |
| 突破 + 缩量 | 结论降级 |
| MACD 金叉 + 缩量 | 弱金叉 |
| 空头排列 + 缩量阴跌 | 下跌中继，勿盲目抄底 |
| 锤头 + 放量 | 反转 K 线质量↑ |
| 上轨假突破 + 放量后缩量回带内 | 偏空 |
| 胀爆 + 价新高 + 主力大幅流出 | 顶部共振，偏空 |
| 抛售高潮 + 长下影 + 翌日不再创新低 | 短期反弹质量↑（非必然反转） |
| 价平量增 + 箱体上沿 | 突破前蓄势或假突破前兆，等放量方向 |

### 常见误判

- **放量即好**：高位放量可能是派发或胀爆
- **缩量即坏**：上升中继缩量回踩属健康
- **只看单日量**：需 3～5 日结构
- **忽略价格位置与趋势阶段**
- **抛售高潮即抄底**：需价格结构确认，可能仅反弹
- **「跳空必回补」**：突破/中继跳空常不补（见 `ta-price-patterns`）

### 失效条件

- 除权、年报季、事件日
- 一字板连续、小盘流动性极差
- 腾讯降级源可能缺 `turnover_rate`
- 涨跌停日量比失真

## 分析步骤

1. 读近 N 日量、换手率、量比；判定当前趋势阶段（初/中/末/横盘）
2. 对照价格方向：价涨量增/价涨量缩/价跌量增等
3. 结合**位置与阶段**查「位置×阶段×量能」表
4. 若存在形态，查形态量能表
5. 与其他迹象组合查表，给出确认/背离/中性
6. 标注数据口径（量比=5日均；主力=东财估算）

## 禁止事项

- 无成交量数据仍下量价结论
- 单日异常量等同趋势反转
- 不区分趋势阶段谈放量

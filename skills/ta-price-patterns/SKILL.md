# 价格形态分析

> **运行时只读本文件。** `modules/skill_loader.load_skill("ta-price-patterns")`

## 何时使用

- 多 K 线结构：头肩、双顶底、三重顶底、三角、楔形、旗形、箱体
- 缺口三类：突破 / 中继（测量）/ 衰竭；岛形反转
- 颈线/边界突破、反扑（回抽）确认与量度目标

## 输入数据

`quotes` OHLCV 序列 ≥60 日；`volume_ratio`；趋势/regime 来自 trend_structure。

## 核心判据

### 反转结构

| 形态 | 成立要点 | 含义 |
|------|----------|------|
| 双重底 | 两底近、第二次缩量、**收市价**破中间高 | 中期偏多 |
| 双重顶 | 两顶近、第二次量衰、**收市价**破中间低 | 中期偏空 |
| 三重底 | 三谷近同水平；**收市价**越过两中间峰连线 | 与头肩类似变体 |
| 三重顶 | 三峰近同水平；各峰量递减；**两中间谷被放量跌破** | 中期偏空 |
| 头肩底 | 右肩>左肩，**收市价**放量破颈线 | 中期偏多 |
| 头肩顶 | 见下节「颈线与反扑」 | 中期偏空 |
| 假破颈线/边界后回到形态内 | — | 形态失效；向上假破常意味原突破不成立 |

**头肩顶要素（原文归纳）**：
1. 事先上升趋势
2. 左肩（A）量重，回调至 B
3. 头上冲（C）量轻创新高
4. 回落至 D，低于 A 且近前低
5. 右肩（E）量显著减轻，不及 C
6. **收市价**明确跌破颈线
7. 反扑回颈线（G）后跌至新低

**正常反弹 ≠ 形态（page-0133）**：价格从前高挡下或从前低弹起属正常；**须收市价破中间位**才确认双重顶/底。有效形态常跨 **2～3 个月**（周线/月线可更长）。

### 圆顶/圆底与 V 形（page-0133）

| 形态 | 要点 | 含义 |
|------|------|------|
| **圆顶/圆底（碟形）** | 趋势缓慢由升转横再转降（或镜像） | 完成时点难精确；量度目标不可靠 |
| 持续时间 | 常现于**周线/月线**，可持续数年 | 越久意义越大 |
| **V 形顶/底** | 突发尖顶/尖底，几乎无过渡 | 最难对付；须等结构确认，勿抢反弹/摸顶 |

### 颈线突破与反扑（消除「破线即确认」歧义）

| 要点 | 规则 |
|------|------|
| 有效突破 | **收市价**穿越颈线/边界，非仅日内刺穿 |
| 初始突破量 | 头肩顶初破时量急增**非必须**；但反扑应量轻，再跌应量重 |
| 反扑 | 破颈线后常反弹至颈线或前低 D；**不一定发生**，可能仅极小反弹 |
| 反扑幅度 | 初破量极大 → 反扑空间小；初破量轻 → 反扑概率大 |
| 反扑后 | 新趋势恢复时应有显著放量 |
| 颈线角色 | 顶破后颈线上方为阻挡；底破后颈线下方为支撑 |

### 持续结构

| 形态 | 成立要点 | 含义 |
|------|----------|------|
| 对称三角 | ≥**4 个转折点**（每边趋势线至少 2 点）；两线聚拢；通常 1～3 个月 | 多为持续；方向随**事前趋势**（上升中偏多向上破） |
| 上升三角 | 上边水平、下边上倾 | 一般看涨 |
| 下降三角 | 下边水平、上边下倾 | 一般看跌 |
| <1 个月小三角 | — | 可能属三角旗，非标准三角 |
| 上升旗形/三角旗 | 整理后向上突破放量 | 延续向上 |
| **旗形 vs 三角旗** | **旗形**：平行通道，旗面与主趋势**反向**倾斜；**三角旗**：两线收敛，更短更尖 | 均属持续形态；突破须放量 |
| 下降旗形 | 向下跌破放量 | 延续向下 |
| 矩形箱体 | 多次测上下沿 | 等突破 |
| **楔形** | 两线聚拢且**均倾斜**，倾角与主流趋势**相反** | **上升楔形**（上倾）→ 看跌；**下降楔形**（下倾）→ 看涨 |
| 扩大形（喇叭） | 两线分离；量随摆幅扩大 | 多现于主要顶部，偏看跌 |
| 破箱体上沿 + 量比>1.5 | — | 偏多 |
| 突破无量 | — | 假突破概率高 |

**三角完成**：收市价穿越任一侧趋势线；突破后该线变支撑/阻力；顶点亦重要；形成过程中量应萎缩，突破时量应明显放大。

### 缺口（三类 + 岛形）

**定义**：相邻 K 线无重叠空白区。向上跳空=坚挺；向下=疲软。「跳空总会被填回」**不正确**——是否回补取决于类型。

| 类型 | 特征 | 回补 | 含义 |
|------|------|------|------|
| **突破跳空** | 形态完成/重要趋势线突破；常伴大额量 | **常不完全回补**；完全填回且回到跳空下方 → 原突破可能不成立 | 向上跳空后常作支撑；向下作阻挡 |
| **中继跳空（测量跳空）** | 趋势中途 | 通常不补 | 趋势延续；可作量度 |
| **衰竭跳空** | 趋势末段 | 随后常快速回补 | 趋势力竭预警 |
| **岛形反转** | 向上衰竭跳空 + 约一周后向下突破跳空，中间孤立 | — | 顶部岛形反转；底部镜像 |

- 突破跳空：量越大，被填回可能性越小。
- 周线/月线缺口意义大于日线。

### 组合

| 组合 | 解读 |
|------|------|
| 颈线突破 + 主力流入 + MACD 同向 | 质量高 |
| 双重底 + 启明星 | 相互验证 |
| 三角 + range 环境 | 突破需放量 |
| 上升楔形于上升趋势 | 偏空预警（与「上升三角」不同：楔形两线均斜且与趋势反向） |
| 突破跳空 + 形态高度量度 | 目标更可靠 |
| 衰竭跳空 + 振荡预备区 | 反转质量↑ |
| 反扑颈线失败 + 量重 | 形态确认加强 |

### 量度

- 通用：自突破点向突破方向投影**形态高度**（头—颈线、箱体高、三角最宽底边等）。
- 对称三角：自突破点或顶点量出与底边相等的距离；亦可用平行于下边 trendline 的管道线作目标。
- 中继跳空：有时可作「测量跳空」，投影约等于缺口前一段趋势幅度。

### 常见误判

- 形态未完成即确认（须等收市价破界）
- 突破无成交量
- 短于 15 根 K 线的大结构
- 忽略右肩/第二顶成交量变化
- 混淆上升三角（上边水平）与上升楔形（两线均斜、与趋势反向）
- **把正常挡回前高当成双重顶**（须破中间低）
- 把所有跳空当「必回补」
- **V 形/圆顶无过渡即下定论**

### 失效条件

- 形态破坏（假突破回形态内）
- 数据不足 20～60 根（视形态而定）
- 仅日内刺穿颈线，收市未确认

## 分析步骤

1. 读 ≥60 日 K 线
2. `modules/patterns.analyze_price_patterns` 得雏形标签
3. 对照上表判确认/形成中/失败；三角查是否 ≥4 转折点
4. 检查突破是否为**收市价**；评估反扑与量能
5. 若有缺口，分类并判回补预期
6. 结合量能、趋势、资金给目标与失效位

## 禁止事项

- 未完成形态当已确认
- 无量突破下强结论
- 日内刺穿当有效突破

# 段永平视角（生意本质）

## 输入
- `research_reports.recent_reports`（行业、标题、机构观点）
- 结构化财报摘要（待 `fundamentals` 模块）
- `symbols` 技术侧仅作环境约束

## 输出评分卡
- score 1～5
- stance: 好生意 / 一般 / 差生意
- bullets: 2～3 条
- falsify: ≥1 条证伪
- confidence: A/B/C

## 判据
- 毛利率与同行比是否占优
- 研报是否强调可复制商业模式
- 管理层是否可信（研报措辞、历史资本配置）
- 数据不足标灰色，禁止编造

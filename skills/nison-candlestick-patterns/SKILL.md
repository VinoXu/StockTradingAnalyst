# 尼森蜡烛图形态判据

> **运行时只读本文件。** `modules/skill_loader.load_skill("nison-candlestick-patterns")`

## 何时使用

- 识别 1～5 根 K 线反转或持续形态
- 必须先定趋势与位置（同形异义：锤子 vs 上吊）
- 与 `ta-candlestick`（墨菲+预备区版）并列参照，不互相替代

## 输入数据

`quotes`：`open`, `high`, `low`, `close`, `volume`

## 核心判据

### 计量定义（判形前先算）

对每根 K 线由 OHLC 计算（按顺序核对，不可跳步）：

| 符号 | 算式 | 用途 |
|------|------|------|
| body | abs(close − open) | 实体大小 |
| upper | high − max(open, close) | 上影线 |
| lower | min(open, close) − low | 下影线 |
| range | high − low（若=0→无效K线） | 振幅 |
| mid | (open + close) / 2 | 实体中点 |
| 阳线 | close > open | 白/空心 |
| 阴线 | close < open | 黑/实心 |
| 长实体 | body ≥ 0.5 × range | 趋势/星形第一根 |
| 小实体/纺锤 | body ≤ 0.33 × range | 星线、孕线 |
| 十字 | body ≤ 0.1 × range | 平衡、不可单独定方向 |
| 向上跳空 | 当日 low > 前日 high | 窗口、星线向上 |
| 向下跳空 | 当日 high < 前日 low | 窗口、星线向下 |
| 实体吞没 | max今(o,c) ≥ max前(o,c) 且 min今(o,c) ≤ min前(o,c) | 吞没 |
| 穿入实体上半 | close ≥ 前 mid | 刺透、启明星第三根 |
| 穿入实体下半 | close ≤ 前 mid | 乌云、黄昏星第三根 |
| 开盘于实体内 | 前 min(o,c) ≤ 今 open ≤ 前 max(o,c) | 三乌鸦、孕线 |

**判形流程**：①定趋势与高低位 → ②对候选K线算上表 → ③逐条核对形态表①②③… → ④查位置表 → ⑤需确认形态检查次日K

### 分析前提

| 规则 | 含义 |
|------|------|
| 趋势变化≠立刻反向 | 反转形态指此前趋势可能变化，未必马上掉头 |
| 主趋势过滤 | 顺主趋势才宜依信号开新仓；逆主趋势多作减仓/了结 |
| 先定趋势再辨形态 | 上升趋势勿把看涨反转当顶；下降趋势反之 |
| 反转 vs 持续 | 多数形态判转折；三法、窗口、并列等判延续 |
| 持续须顺势 | 看涨持续只在上升趋势；看跌持续只在下降趋势 |
| 伞形线同形异义 | 下跌后=锤子（偏多）；上涨后=上吊（偏空） |
| 理想形态少见 | 允许主观判断；标注置信度 |

### 单根基石（计量标准）

| 类型 | 判定条件 | 含义 |
|------|----------|------|
| 长实体 | body ≥ 0.5×range | 方向明确；组合第三根+放量质量↑ |
| 短实体/纺锤 | body ≤ 0.33×range | 犹豫；星线、孕线基础 |
| 十字线 | body ≤ 0.1×range | 平衡；**不可单独定方向** |
| 光脚阳线 | 阳线且 lower=0 | 捉腰带看涨构件 |
| 光头阴线 | 阴线且 upper=0 | 捉腰带看跌构件 |

### 看涨反转（须低位/支撑/下跌末段）

| 形态 | 结构要点 | 含义 |
|------|----------|------|
| 锤子（蜡烛）线 | ①前提：下跌趋势或近10根呈低位/支撑区 ②K1小实体：body≤0.33×range，实体位于range上1/3（min(o,c)距high较近） ③下影：lower≥2×body ④上影：upper≤0.1×range（可无） ⑤四条件全满足→锤子；同形在上涨末段→改判上吊 | 低位反转偏多 |
| 倒锤子（蜡烛）线 | ①前提：下跌趋势 ②K1小实体在range下1/3 ③上影：upper≥2×body ④下影：lower≤0.1×range ⑤确认（须下一根K2）：K2.close>max(K1.o,K1.c)；未确认→标注「待确认」 | 潜在见底；须次日收高于实体确认 |
| 启明星形态 | ①前提：下跌趋势 ②K1：阴线长实体（body≥0.5×range，close<open） ③K2：小实体（body≤0.33×range），且向下跳空（K2.high<K1.low） ④K3：阳线长实体，close≥K1.mid 且 close>min(K1.o,K1.c)（穿入K1实体） ⑤K1-K2-K3顺序满足→启明星 | 底部反转偏多 |
| 十字启明星形态 | ①满足启明星①～④ ②K2为十字线（body≤0.1×range）而非纺锤 ③其余同启明星 | 底部反转意味更强 |
| 刺透形态 | ①前提：下跌趋势 ②K1：阴线（close<open） ③K2：阳线（close>open） ④K2.open<K1.low（开盘低于前日最低） ⑤K2.close>K1.mid 且 K2.close<max(K1.o,K1.c)（升入前阴实体上半但未完全吞没） ⑥⑤成立→刺透；若K2实体完全包住K1→改判看涨吞没 | 底部反转偏多 |
| 看涨吞没形态 | ①前提：下跌趋势 ②K1：阴线 ③K2：阳线 ④实体吞没：K2.open≤K1.close 且 K2.close≥K1.open （今开收区间完全覆盖前开收区间） ⑤④成立→看涨吞没 | 底部反转偏多 |
| 平底锅底部形态 | ①多根K线形成圆底（高低点逐步抬高、振幅收窄） ②出现向上跳空窗口（当日low>前日high）作为验证 ③缺窗口→仅标注「圆底待验证」 | 底部反转偏多 |
| 弃婴形态 | ①三根K：K1长实体、K2十字（body≤0.1×range）、K3长实体 ②K2与K1、K3均跳空（K2.low>K1.high 或 K2.high<K1.low，含影线） ③三条件全满足→弃婴（罕见） | 罕见强反转 |
| 捉腰带线 | 看涨：K1阳线长实体（body≥0.5×range），open=low（无上影/光脚），处于低位区 看跌：K1阴线长实体，open=high（无下影/光头），处于高位区 | 低位阳线偏多/高位阴线偏空 |

### 看跌反转（须高位/阻力/上涨末段）

| 形态 | 结构要点 | 含义 |
|------|----------|------|
| 上吊（蜡烛）线 | ①前提：上涨趋势或近10根呈高位/阻力区 ②K1结构同锤子：body≤0.33×range、lower≥2×body、upper≤0.1×range ③确认（须K2）：K2.open<max(K1.o,K1.c) 且 K2.close<min(K1.o,K1.c)（理想） ④②满足但③未满足→「上吊待确认」 | 高位反转偏空；须次日收低于实体确认 |
| 乌云盖顶 | ①前提：上涨趋势 ②K1：阳线（close>open） ③K2：阴线（close<open） ④K2.open>K1.high ⑤K2.close<K1.mid 且 K2.close>min(K1.o,K1.c)（跌入前阳实体下半） ⑥④⑤成立→乌云盖顶 | 顶部反转偏空 |
| 黄昏星形态 | ①前提：上涨趋势 ②K1：阳线长实体 ③K2：小实体，向上跳空（K2.low>K1.high） ④K3：阴线，close≤K1.mid 且 close<max(K1.o,K1.c)（穿入K1实体） ⑤③④成立→黄昏星 | 顶部反转偏空 |
| 十字黄昏星形态 | ①满足黄昏星①～④ ②K2为十字线（body≤0.1×range） ③其余同黄昏星 | 顶部反转意味更强 |
| 看跌吞没形态 | ①前提：上涨趋势 ②K1：阳线 ③K2：阴线 ④K2.open≥K1.close 且 K2.close≤K1.open（实体完全吞没） ⑤④成立→看跌吞没 | 顶部反转偏空 |
| 三只乌鸦 | ①前提：高位或上涨末段 ②K1、K2、K3连续三根阴线 ③各根body≥0.4×各自range（长实体） ④各根close距low≤0.15×range（收近最低） ⑤K2.open介于K1实体内，K3.open介于K2实体内 ⑥close逐根降低：K2.close<K1.close，K3.close<K2.close ⑦全满足→三只乌鸦 | 高位顶部反转偏空 |
| 流星（蜡烛）线 | ①前提：上涨趋势 ②K1小实体：body≤0.33×range，实体在range下1/3 ③上影：upper≥2×body ④下影：lower≤0.1×range ⑤四条件全满足→流星 | 上攻失败，偏空 |
| 墓碑十字线 | ①body≤0.1×range ②open≈close≈low（收开均在最低，upper为唯一长影） ③upper≥2×body ④前提：高位区 | 高位偏空 |
| 圆形顶部形态 | ①多根K线形成圆顶（高点弧线、振幅收窄） ②出现向下跳空窗口（当日high<前日low）验证 ③缺窗口→「圆顶待验证」 | 顶部反转偏空 |
| 前方受阻形态 | ①先出现白色三兵（连续三根阳线） ②第2、3根出现衰竭：上影变长（upper>body）或实体缩小（body<K1.body） ③②在③基础上成立→前方受阻 | 上涨动能衰竭，偏空 |
| 向上跳空两只乌鸦 | ①K1：阳线长实体 ②K2：阴线，向上跳空（K2.low>K1.high） ③K3：阴线，open>K2.open 且 close<K2.close ④三根顺序满足→向上跳空两只乌鸦（罕见） | 罕见顶部反转 |
| 三山顶部形态 | ①较长周期内三次冲高 ②三次高点落在同一价格带（偏差≤1.5%） ③每次冲高后回落 ④③次成立→三山顶 | 较长期顶部反转偏空 |

### 其他反转

| 形态 | 结构要点 | 含义 |
|------|----------|------|
| 吞没形态 | 看涨：下跌后，K2阳包K1阴（K2.open≤K1.close且K2.close≥K1.open） 看跌：上涨后，K2阴包K1阳（K2.open≥K1.close且K2.close≤K1.open） | 下跌后偏多/上涨后偏空 |
| 反击（蜡烛）线 | ①K1、K2颜色相反 ②K2开盘跳空：下跌后反弹→K2.open<K1.low；上涨后回落→K2.open>K1.high ③K2.close≈K1.close（差值≤0.3%或≤0.01×价格） ④②③成立→反击线 | 趋势力量衰减，潜在反转 |
| 孕线形态 | ①K1：长实体（body≥0.5×range） ②K2：小实体（body≤0.5×K1.body） ③K2完全处于K1实体内：max(K2.o,K2.c)≤max(K1.o,K1.c) 且 min(K2.o,K2.c)≥min(K1.o,K1.c) ④③成立→孕线 | 趋势暂停，潜在反转 |
| 十字孕线形态 | ①满足孕线①～③ ②K2为十字（body≤0.1×range） ③②成立→十字孕线 | 反转意味更强 |
| 切入线形态 | ①前提：下跌 ②K1：阴线 ③K2：阳线小实体 ④K2.close略高于K1.close（在K1实体内部） ⑤失效：跌破K2.low→恢复下跌 | 持续看空；跌破白线低点恢复下跌 |
| 待入线形态 | ①前提：下跌 ②K1：阴线 ③K2：阳线小实体 ④K2.close≈K1.low（差≤0.2%） ⑤④成立→待入线 | 持续看空 |
| 插入线形态 | ①前提：下跌 ②K1：阴线 ③K2：阳线 ④K2.close>min(K1.o,K1.c) 且 K2.close<K1.mid（穿入前阴实体但未过中点） ⑤弱于刺透；下跌中偏空 | 弱于刺透；下跌中偏空 |
| 三星形态 | ①连续三根K均为星线（body≤0.33×range） ②三根均从前长实体处跳空 ③整体构成启明/黄昏星框架 ④极罕见 | 罕见强反转 |
| 平头顶部形态和平头底部形态 | 平头顶：相邻两日high相等（差≤0.3%） 平头底：相邻两日low相等（差≤0.3%） 可跨2～5日；与其他反转形态共振时加重 | 小规模反转；与其他形态共振时加重 |
| 伞形线 | ①body≤0.33×range ②lower≥2×body ③实体在range上1/3 ④下跌后→锤子语义；上涨后→上吊语义 | 下跌后偏多/上涨后偏空，看位置 |
| 三尊形态 | 三尊顶：三个高点，中间峰最高（类头肩顶） 倒三尊/三川：三个低点，中间谷最深（类倒头肩） 须结合趋势与位置 | 顶部或底部反转；须看位置 |
| 三川底部形态 | ①较长周期三次下探 ②三次低点在同一支撑带（偏差≤1.5%） ③每次下探后反弹 ④③次成立→三川底 | 较长期底部反转偏多 |
| 大风大浪线 | ①body≤0.33×range ②upper≥range×0.35 且 lower≥range×0.35（上下影均很长） ③单根出现→方向性丧失 | 原方向性丧失；犹豫/转折警惕 |
| 白色三兵形态或者白色三兵挺进形态 | ①K1、K2、K3连续三根阳线 ②close逐根抬高：K2.close>K1.close，K3.close>K2.close ③各根close距high≤0.2×range（收近最高） ④全满足→白色三兵 | 底部或中继偏多 |
| 塔形形态 | 塔顶：①左侧1～数根阳线长实体 ②中间横盘小实体 ③右侧1～数根阴线长实体，跌破整理区 塔底：镜像（先阴、整固、后阳突破） | 塔顶偏空/塔底偏多 |
| 星（蜡烛）线 | ①body≤0.33×range（纺锤/小实体） ②从前长实体跳空：向上跳空→K.low>前.max(o,c)；向下跳空→K.high<前.min(o,c) ③常作启明星/黄昏星中间根 | 星线；启明/黄昏星中间根 |

### 持续形态（须顺势）

| 形态 | 结构要点 | 含义 |
|------|----------|------|
| 三法形态 | ①K1：长实体顺势（阳或阴） ②K2-K4：三根逆色小实体，high≤K1.high 且 low≥K1.low（完全在K1范围内） ③K5：长实体顺势，收盘突破K1极值（上升：close>K1.high；下降：close<K1.low） ④①～③+顺势环境→三法 | 持续形态，须顺势 |
| 上升三法形态 | ①上升趋势 ②K1长阳 ③K2-K4三小阴在K1实体内 ④K5长阳收盘>K1.high ⑤④成立→上升三法 | 上涨趋势中持续看多 |
| 下降三法形态 | ①下降趋势 ②K1长阴 ③K2-K4三小阳在K1实体内 ④K5长阴收盘<K1.low ⑤④成立→下降三法 | 下跌趋势中持续看空 |
| 窗口 | 向上窗口：当日low>前日high（跳空缺口）→支撑区，偏多持续 向下窗口：当日high<前日low→阻挡区，偏空持续 窗口未回补前保持支撑/阻挡效力 | 持续信号；窗口作支撑/阻挡 |
| 分手线 | ①K2.open=K1.open（同开盘价，差≤0.1%） ②K1、K2颜色相反 ③上升环境：K2.close>K1.close；下降环境：K2.close<K1.close ④③成立→原趋势将恢复 | 原趋势将恢复 |
| 并列阳线 | ①K1、K2均为阳线 ②open相同（差≤0.1%） ③body高度接近（差≤30%） ④向上跳空后并列→持续看多；向下跳空后并列→持续看空 | 持续形态 |
| 跳空并列阴阳线形态 | 上升：①向上跳空阳线K1 ②K2阴线，open在K1实体内，close<K1.close ③K2.close为关键位 下降：①向下跳空阴线 ②K2阳线，open在K1实体内，close>K1.close | 持续；收市价是关键位 |
| 高价位跳空突破形态 | ①前期急涨 ②近新高处连续小实体整固（振幅收窄） ③从整固区向上跳空（low>整固区high） ④③成立→高价位跳空突破 | 看涨持续 |
| 低价位跳空突破形态 | ①前期急跌 ②近新低处连续小实体整固 ③从整固区向下跳空（high<整固区low） ④③成立→低价位跳空突破 | 看跌持续 |

### 十字线家族

| 形态 | 结构要点 | 含义 |
|------|----------|------|
| 十字（蜡烛）线 | ①body≤0.1×range（open与close几乎相等） ②单独出现不可定方向 ③须结合位置与其他形态 | 平衡；不可单独定方向 |
| 十字星（蜡烛）线 | ①满足十字线① ②从前长实体跳空（low>前high 或 high<前low） ③作星线/启明/黄昏中间根 | 星线；反转或持续视环境 |
| 蜻蜓十字线 | ①body≤0.1×range ②open≈close≈high（无上影） ③lower≥2×body ④低位区→偏多 | 低位偏多 |
| 长腿十字线 | ①body≤0.1×range ②upper≥0.35×range 且 lower≥0.35×range ③开收在range中部→黄包车夫线 | 市场失方向；中部为黄包车夫线 |
| 北方十字线 | 上涨行情中出现 | 高位谨慎 |
| 南方十字线 | 下跌行情中出现 | 低位谨慎 |

### 位置 × 形态

| 位置 | 典型形态 | 解读 |
|------|----------|------|
| 下跌末段+支撑 | 锤子、刺透、吞没、启明星 | 质量高 |
| 上涨末段+阻力 | 上吊、乌云、吞没、黄昏星 | 质量高；上吊须次日确认 |
| 横盘中部 | 任意反转 | 降权或忽略 |
| 上涨中段 | 锤头形似 | 多为中继，非反转 |
| 涨跌停/一字 | 影线失真 | 不硬判 |

### 常见误判

- 孤立形态不看位置（锤头/上吊同形）
- 十字线单独定方向
- 吞没/刺透/乌云无趋势与位置前提
- 上吊无次日验证仍当顶部确认
- 持续形态出现在逆趋势环境
- 罕见形态当理想形态已确认

### 失效条件

- 无 OHLC 或根数不足
- 横盘中部低权重
- 缺确认条件的形态（标注「待确认」）

## 分析步骤

1. 定主趋势与高低位（上升/下降/横盘；是否近支撑/阻力）
2. 对近 5～10 根 K 线逐根计算 body/upper/lower/range/mid
3. 按「计量定义」算完后，对照形态表逐条核对①②③…（全部满足才认定）
4. 需确认形态（上吊、倒锤子等）检查次日 K 是否满足确认条件
5. 查位置表过滤；横盘中部降权
6. 多信号同区 → `nison-signal-confluence`；需西方验证 → `nison-ta-integration`

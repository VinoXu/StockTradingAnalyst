# 尼森蜡烛图 × 西方技术验证判据

> **运行时只读本文件。** `modules/skill_loader.load_skill("nison-ta-integration")`

## 何时使用

- 蜡烛图信号需趋势线、回撤、均线、振荡指标、量或目标位验证
- 与 `ta-trend-structure`、`ta-oscillators`、`ta-volume-price` 等并列，提供尼森联用规则

## 输入数据

`quotes`、`indicators`（均线、RSI、KDJ、MACD 等）、`volume`；趋势线/回撤由价格序列判定

## 核心判据

### 趋势线

| 规则 | 含义 |
|------|------|
| 上升支撑线 | 连接抬升低点（下影低点）；线上买方占优 |
| 下降阻挡线 | 连接降低高点（上影高点）；线下卖方占优 |
| 破低反涨 | 短暂跌破支撑后迅速收回支撑上方 → 看涨 |
| 破高反跌 | 短暂突破阻挡后跌回阻挡下方 → 看跌 |
| 极性转换 | 原支撑跌破后变阻挡；原阻挡突破后变支撑 |

### 百分比回撤

| 水平 | 含义 |
|------|------|
| 50% | 最常见回撤位 |
| 38.2% / 61.8% | 斐波那契回撤；与蜡烛反转信号共振时加重 |
| 回撤位+蜡烛形态 | 须相互验证，不单凭回撤下单 |

### 移动平均线

| 规则 | 含义 |
|------|------|
| 价格>均线 | 趋势偏多（参数随周期而定） |
| 均线作支撑/阻挡 | 收市价有效突破/跌破须确认 |
| 黄金交叉 | 短均线上穿长均线 → 偏多 |
| 死亡交叉 | 短均线下穿长均线 → 偏空 |
| 横盘时均线 | 交叉/触轨失效率高，降权 |

### 摆动指标与蜡烛图

| 规则 | 含义 |
|------|------|
| 负面背离+看跌蜡烛 | 顶部反转加强 |
| 正面背离+看涨蜡烛 | 底部反转加强 |
| RSI 进超买+上吊/黄昏星 | 多头宜减仓观察 |
| RSI 进超卖+锤子/启明星 | 空头宜谨慎 |
| 摆动指标 | 须与价格环境一起读，不单指标定方向 |

### 交易量

| 规则 | 含义 |
|------|------|
| 突破+放量 | 真突破概率↑ |
| 突破+缩量 | 降权或观望 |
| 反转形态+放量 | 质量↑（如吞没、刺透） |
| 上涨日量>回调日量 | 趋势健康 |
| 高位十字/纺锤+巨量 | 抛压/转折警惕 |

### 价格目标

| 规则 | 含义 |
|------|------|
| 箱体突破 | 目标≈箱体高度，自突破点投影 |
| 对等运动 | 调整后再沿原方向，幅度不少于第一波 |
| 旗形/尖旗形 | 整理后延续；突破方向量度 |

### 东西方相互验证

| 规则 | 含义 |
|------|------|
| 蜡烛反转+西方形态/指标同向 | 置信度显著提高 |
| 蜡烛信号被西方工具否定 | 观望或降权 |
| 缺西方验证数据 | 标注「未验证」，禁止编造 |

## 分析步骤

1. `nison-candlestick-patterns` 识别形态
2. 按问题选上表维度验证（趋势/回撤/均线/摆动/量/目标）
3. 记录汇聚或背离
4. 输出时写明验证了哪些维度

## 禁止事项

- 缺数据仍写「已相互验证」
- 用本 Skill 替代各 `ta-*` 的细部计算定义

# 摆动指标分析

> **运行时只读本文件。** `modules/skill_loader.load_skill("ta-oscillators")`

## 何时使用

- 解读 MACD / RSI / KDJ / **CCI / Williams %R**
- 零轴、超买超卖、金叉死叉、钝化、背离
- **ROC/动力指数**类变化速率概念（本地多由 MACD/RSI 覆盖）
- **ADX** 判别趋势市 vs 横盘（`indicators.adx14`；缺失时看 regime）
- 为 `ta-candlestick` **反转形态**提供预备区过滤（第 12 章）
- **反意见理论**：极端一致预期 → 环境降权（非 A 股自动数据）

## 输入数据

`indicators`：`macd`, `macd_signal`, `macd_hist`, `rsi14`, `k`, `d`, `j`, **`cci20`, `williams_r14`, `adx14`, `plus_di14`, `minus_di14`**；`quotes`：`close`, `volume`  
参数：MACD 12/26/9，RSI 14，KDJ 9/3/3（J=3K−2D），CCI 20，Williams %R 14，ADX 14。**D 线**在过滤链中对应慢速随机指标的 **%D**（本地 KDJ 的 `d` 字段）。

## 核心判据

### MACD

| 迹象 | 含义 |
|------|------|
| DIF 上穿 DEA，均在零轴上 | 零轴上金叉，动量偏多 |
| 零轴附近金叉，柱体由负转正并放大 | 转折质量较好 |
| 零轴下方远离零轴金叉 | 弱反弹，失败率高，需放量+均线确认 |
| 零轴上方死叉 | 可能是洗筹，勿机械看空 |
| 零轴下方死叉 + 柱体扩大 | 动量偏空 |
| 柱体连续放大 | 动量强化 |
| 柱体缩短 | 动量衰减 |
| 价新高 MACD 高点降低 | 顶背离 |
| 价新低 MACD 低点抬高 | 底背离 |

### RSI

| 迹象 | 含义 |
|------|------|
| RSI>70 | 超买；强趋势中可钝化，勿机械卖 |
| RSI<30 | 超卖；下跌中继可反复超卖 |
| 上升趋势 RSI 40～50 再起 | 强势区健康回调 |
| 下降趋势 RSI 50～60 受阻 | 弱势反弹 |
| 价新低 RSI 抬高 / 价新高 RSI 走低 | 底背离 / 顶背离 |
| **RSI 顶/底衰竭** | 超 70 后峰无力创新高再破前低=顶；超 30 后谷无力创新低再破前高=底 |

### KDJ（含慢速随机 %K/%D 逻辑）

| 迹象 | 含义 |
|------|------|
| K、D<20 金叉 | 低位金叉，偏反弹 |
| K、D>80 死叉 | 高位死叉，偏回调 |
| K、D>80 长期粘连 | 高位钝化 |
| J>100 拐头向下 / J<0 拐头向上 | 极端区转折风险/机会 |
| 同金叉：K≈30 vs K≈70 | 前者偏多，后者仅弱修复 |

### CCI（商品通道指数，Lambert）

- 构造：当前价 vs **20 日 MA**，按平均偏差标准化；刻度 **+100～−100** 恒定。
- **两种读法（勿混）**：
  - **趋势跟随**（Lambert 原意）：CCI **上穿 +100** 才考虑做多；**下破 −100** 才考虑做空。
  - **超买超卖**（业内常用）：**>+100** 超买；**<−100** 超卖；极端区背离 → 反转预警。
- 与 RSI/KDJ 同：须服从 `ta-trend-structure` 趋势；强趋势中可长期贴轨。
- **预备区过滤**（第 12 章，近 20 日窗口同 D/RSI）：近 20 日内 **曾 <−100**（或当前 <−100）→ 看涨反转 K；**曾 >+100**（或当前 >+100）→ 看跌反转 K。
- 本地字段 **`indicators.cci20`**；`sync_symbol` / `refresh_indicators` 后可用；缺失时标注「CCI 未验证」。

### Williams %R

- 概念同随机指标：收市价在过去 N 日区间中的相对位置；公式从「最高价−收价」出发，**软件常颠倒刻度**使读法类似 KDJ。
- **>80** 超买、**<20** 超卖（与随机 80/20 对称）；极端区 **背离** 为重点。
- 预备区：近 20 日内 **曾 <20** / **曾 >80**，逻辑与 %D 相同，可替代 D 作蜡烛过滤。
- 本地字段 **`indicators.williams_r14`**（0～100 刻度）；缺失时用 KDJ 的 D 或 RSI 代替。

### 快速 vs 慢速随机（概念）

| 类型 | 特点 | 本地映射 |
|------|------|----------|
| **快速随机** | %K 更敏感，交叉频繁 | KDJ 的 `k` |
| **慢速随机** | %D 平滑 %K，预备区过滤主用 | KDJ 的 `d` |
| **日 + 周组合** | 方向看周、时机看日 | 周级需人工或后续多周期 sync |

### ROC / 动力指数（变化速率）

| 类型 | 要点 |
|------|------|
| **动力指数** | 价格**差值**；急涨急跌后易扭曲，须平滑；有零线，零上/零下表方向 |
| **ROC** | 价格**比值** `100×(C/Cx)`；**100 为零线**；>100 价高于 x 日前，<100 反之 |
| **零线穿越** | 上穿零线=买预警、下穿=卖预警；**仅当与主趋势同向**才有效（上升中才采信上穿） |
| **与 MACD** | MACD = 双 EMA 差 + 平滑，可视为改进版变化速率；本地优先用 MACD 柱/零轴 |

- 动力/ROC 在**极端区转折常先于价格**（page-0222～0223）；须配合趋势，不可孤立逆大势。

### MACD 刷形图（柱体）

- 刷形图 = DIF 与 DEA **差距**；穿零 = 买卖信号；**转折常早于** DIF/DEA 交叉（page-0243）。
- 本地 `macd_hist` 可近似刷形图：柱体缩短/转向 = 动量衰减/转折预警。

### 预备区（第 12 章 → 服务蜡烛图过滤）

随机/%D 进入极端区后，迟早产生交易信号；**>80 与 <20 为信号预备区**，%D **先**进入预备区，才配合蜡烛反转形态。

**「曾」的时间窗口（必遵，避免误读）**

- **不是**历史上任意一次（如一年前 D>80 **不算**）。
- **仅指本轮价格 swing 内、形态出现前最近一段**：查近 **20 个交易日**（约 1 个月）的 D/RSI 序列；若无 OHLC 序列则看近 **20 根 K 线**。
- 预备区进入、折返、蜡烛反转应发生在**同一段短期走势**（通常相隔数根 K 线～数周）；中间长期横盘、D 长期远离极端区后出现的 K 线 → **预备区失效**，按「未进预备区」忽略。
- 程序/人工：取 `d`、`rsi14` 近 20 日 min/max；近 20 日内未触过 <20/>80（或 RSI 30/70）→ 不算「曾进入」。

| 预备区 | 条件 | 配合的蜡烛反转 |
|--------|------|----------------|
| **看多预备区** | 近 20 日内 D **曾 <20**，或**当前** <20 | 锤头、刺透、启明星、看涨吞没等 |
| **看空预备区** | 近 20 日内 D **曾 >80**，或**当前** >80 | 上吊、乌云盖顶、黄昏星、三只乌鸦等 |
| **买入信号形成** | 近 20 日内 D **曾 <20**，且**当前已升回 >20** | 看涨反转 K 可采信 |
| **卖出信号形成** | 近 20 日内 D **曾 >80**，且**当前已从高位回落**（仍 >80 或已跌破 80 均可，须与形态同 swing） | 看跌反转 K 可采信 |
| **未进预备区** | 如近 20 日 D 始终在 20～80，或 D=65 出现反转 K | **忽略**蜡烛反转（例：page-0286） |

**CCI / %R 预备区**（近 20 日，逻辑相同）：

| 指标 | 看多预备 | 看空预备 |
|------|----------|----------|
| CCI | 曾 **<−100** 或当前 <−100 | 曾 **>+100** 或当前 >+100 |
| Williams %R | 曾 **<20** 或当前 <20 | 曾 **>80** 或当前 >80 |

RSI 过滤时窗口相同：近 20 日内曾 <30 / >70，逻辑与 D 一致。

### 趋势 × 振荡（第 10 章）

| 原则 | 含义 |
|------|------|
| 趋势第一 | 先定主要趋势，再用振荡找时机；**不可**因背离 alone 逆大势 |
| 上升趋势 | 超卖区买信号更灵；超买可钝化，勿机械空 |
| 下降趋势 | 超买区卖信号更灵；超卖勿盲目抄底 |
| 上升+超卖 | 顺势做多时机 |
| 下降+超买 | 顺势做空/减仓时机 |

### 振荡何时最有效 / 何时暂弃

| 市场 | 振荡指标 | 处理 |
|------|----------|------|
| **横盘延伸** | 峰谷与价格几乎同步 | 振荡指标**最有效**，权重↑ |
| **重要价格突破**（新趋势早期） | 已处超买/超卖 | **暂弃振荡**，跟价格突破；勿因超买拒买、因超卖拒卖 |
| 震荡市 | MACD 频繁交叉 | 振荡权重↑，趋势信号↓ |

### ADX 与趋势/横盘判别（第 15 章，概念）

本地 **`indicators.adx14`, `plus_di14`, `minus_di14`**；`analyze_oscillators` 输出 `market_mode`。若无 ADX 字段，用 `ta-trend-structure` 的 **regime** 近似：

| ADX 行为 | 含义 | 指标权重 |
|----------|------|----------|
| 从 **20 以下** 向上 | 市场重新进入**趋势态** | MA/趋势跟随↑，振荡↓ |
| **上升中** | 趋势性加强 | 顺势系统（MA、MACD 零轴）更有效 |
| 从 **40 以上** 回落 | 趋势性减弱，或入横盘 | 振荡指标权重↑ |
| ADX 低且横盘 | 无趋势 | 振荡有效；趋势信号降权 |

+DI / −DI：+DI 上穿 −DI 偏多，下穿偏空；可作敏感趋势系统的过滤器（本地可选）。

### 抛物线 SAR（第 15 章 · 概念层，本地无字段）

Wilder 抛物线（SAR）= **时间×价格**反转跟踪系统，与 ADX/DMI 同章；**始终在市**的止损/反手工具。

| 要点 | 含义 |
|------|------|
| SAR 点在价下 | 多头跟踪；上穿 SAR → 翻空预警 |
| SAR 点在价上 | 空头跟踪 |
| 与 ADX | ADX 判有无趋势；SAR 判跟踪止损，**不替代**趋势结构 |

**本地局限**：无 `sar` 字段。用 **ADX + 趋势线/均线** 近似；**不得**单独依据 SAR 给买卖指令。待工程化后可接入。

### 反意见理论（第 10 章 · 环境层，非个股自动信号）

**核心**：当绝大多数参与者已同向建仓，**剩余推动趋势的买盘/卖盘不足**；极端一致时反转概率上升。须与正统技术（支撑/阻力、趋势线、振荡背离）**共同验证**，不可单独逆大势。

**看涨一致数字（期货/美股顾问调查；A 股无标准字段）**：

| 读数 | 含义 |
|------|------|
| 均衡 **~55%** | 含大众天然偏多修正 |
| **>75%** | 危险区，可能见顶 |
| **>90%** 或 **<20%** | 极端，可考虑反向（须等技术确认） |
| **<25%** | 危险区，可能见底 |
| 75%～90% / 25%～20% | 等**数字本身趋势转向**再逆势，勿机械反手 |

**操作原则**：
- 顺一致方向交易，直到达极端 → 再警惕转折。
- 极端区 + **振荡背离** → 参考价值高（page-0246）。
- 超买区利好不涨 / 超卖区利空不跌 → 转折预警。
- 期货：持仓量极高 + 一致极端 → 反向潜力大；但**持仓仍升时勿立反向仓**，等持仓持平或下降（page-0245）。
- **A 股落地**：无 Investor Intelligence / AAII 时，仅作**环境提示**（如媒体一致看多）；**不得**捏造一致指数；可标注「情绪极端，技术结论降权或等确认」。

### 跨指标组合

| 组合 | 解读 |
|------|------|
| 零轴上 MACD 金叉 + 量比>1.2 + MA 非空头 | 偏多质量高 |
| 零轴下 MACD 金叉 + 缩量 + 均线空头 | 弱反弹或中性 |
| RSI 超卖回升 + KDJ 低位金叉 + 在支撑 | 反弹组合 |
| D<20 + 刺透/锤头（蜡烛） | 第 12 章标准过滤链 |
| D>80 + 乌云/黄昏星（蜡烛） | 第 12 章标准过滤链 |
| MACD 顶背离 + RSI>70 衰竭 + KDJ 高位死叉 | 多重顶预警 |
| MACD 与 KDJ 矛盾 | 标注分歧，降置信度 |
| RSI+KDJ 同处超买/超卖 | 信号质量↑（page-0236） |
| 日随机 + 周随机同向 | 方向看周，时机看日 |
| 一致指数>75% + 顶背离 + 阻力 | 环境+技术共振偏空 |
| ADX 升 + 零轴上 MACD 金叉 | 趋势态顺势质量↑ |
| ADX 降 + 横盘 + KDJ 交叉 | 振荡市，交叉可采信度↑ |

### 常见误判

- 金叉即看多（零轴下弱金叉失败率高）
- RSI>70 必卖 / RSI<30 必买（趋势钝化）
- 只看交叉不看柱体与零轴
- **背离即逆大势**（须服从 trend-structure）
- **反转 K 线不过预备区**（近 20 日 D 未进极端区时，D=65 的锤头无效）
- **把很久以前的极端区当「曾」**（如一年前 D>80，中间 D 长期中性 → 无效）
- 突破初期因超买放弃跟势
- **CCI 趋势跟随与超买超卖读法混用**
- **反意见单独反手**（无技术确认）
- **无 ADX/regime 时在横盘机械跟 MACD 交叉**

### 失效条件

- K 线<35 根（MACD）、<15 根（RSI）
- 一字板、重大消息日
- 横盘频繁交叉且无预备区上下文

## 分析步骤

1. 确认 `ta-trend-structure` 趋势方向
2. 读 MACD 结构、RSI 区域、KDJ 位置
3. **标注 D/RSI 是否在预备区**（供蜡烛过滤）
4. 判突破初期 vs 横盘 → 决定是否暂弃超买超卖
5. 查组合表 + regime 约束
6. 输出倾向 + 预备区状态 + 失效说明

## 禁止事项

- 单一指标直接买卖指令
- 缺数据编造数值
- 忽略大盘/板块与趋势方向
- 未进预备区却确认蜡烛反转

# 蜡烛图形态

> **运行时只读本文件。** `modules/skill_loader.load_skill("ta-candlestick")`

## 何时使用

- 1～5 根 K 线反转/持续形态
- 必须结合**趋势方向、位置、成交量**
- **反转形态**须经第 12 章振荡指标「预备区」过滤（见下）

## 输入数据

`quotes`：`open`, `high`, `low`, `close`, `volume`  
`indicators`：`rsi14`, `k`, `d`, `j`（过滤链主用 `d` 或 `rsi14`）

## 核心判据

### 分析前提（第 12 章）

| 规则 | 含义 |
|------|------|
| 先定趋势 | 辨形态前须明确上升/下降/横盘；上升趋势中不宜把看涨反转当顶部信号，反之亦然 |
| 反转 vs 持续 | 绝大多数形态判转折；少数（三法、跳空并列等）判趋势延续 |
| 持续形态须顺势 | 看涨持续仅出现在上升趋势；看跌持续仅出现在下降趋势 |

### 单根 K 线基石

| 形态 | 结构 | 含义 |
|------|------|------|
| 长实体 | 开收差距大 | 方向明确 |
| 纺锤线 | 小实体+长影线 | 犹豫 |
| 十字线 | 开≈收 | 平衡；**不可单独定方向** |
| 蜻蜓十字 | 长下影无上影 | 低位偏多 |
| 墓碑十字 | 长上影无下影 | 高位偏空 |

**长实体 / 短实体（page-0279）**：
- **长实体**（开收差距大）：方向明确；趋势中长阳/长阴验证方向；反转组合里第三根长实体+**放量** → 质量↑。
- **短实体**（开收差距小）：犹豫；星线、孕线、纺锤线的基础。
- 判实体大小时**暂只看开收距，不混影线**；同一形态下，长实体优于短实体。

### 反转形态（须低位/高位/支撑/阻力）

| 形态 | 结构要点 | 含义 |
|------|----------|------|
| 锤头/锤子线 | 长下影≥2×实体，小上影 | **下跌末段/支撑** → 偏多 |
| 上吊线 | 与锤头同形 | **上升末段/阻力** → 偏空（形态同、位置反） |
| 倒锤头 | 长上影在低位 | 潜在见底，需次日确认 |
| 流星线/射击之星 | 长上影≥2×实体 | 高位上攻失败 |
| 看涨吞没 | 后阳包住前阴 | 低位反转 |
| 看跌吞没 | 后阴包住前阳 | 高位反转 |
| **刺透形态** | 前阴+后阳；后阳开盘低于前低，收盘升入前阴**实体上半** | 看涨（乌云盖顶镜像） |
| **乌云盖顶** | 前阳+后阴；后阴开盘高于前高，收盘跌入前阳**实体下半** | 看跌 |
| **孕线** | 后一根小实体完全处于前一根实体内 | 趋势暂停/潜在反转 |
| **捉腰带线** | 长实体、一端无影线（开盘即最高/最低） | 低位阳线偏多/高位阴线偏空 |
| **反击线** | 两根K收价相同、颜色相反 | 潜在反转，需位置+预备区 |
| 十字孕线 | 孕线+十字 | 反转意味更强 |
| 启明星 | 大阴+星线（跳空小实体）+大阳；第三阳深入第一阴实体 | 底部转折 |
| **十字启明星** | 启明星+中间十字/纺锤 | 反转意味更强 |
| 黄昏星 | 大阳+星线+大阴；第三阴深入第一阳实体 | 顶部转折 |
| **十字黄昏星** | 黄昏星+中间十字 | 反转意味更强 |
| **三只乌鸦** | 三根连续阴线，各收低于前收，开盘在前一根实体内 | 顶部反转 |
| **白色三兵/三白兵** | 三根连续阳线，逐步抬高 | 底部/中继偏多 |

### 持续形态

| 形态 | 含义 |
|------|------|
| 上升三法 | 长阳+3 小阴（均在其内）+ 突破长阳；上升趋势中 |
| 下降三法 | 镜像；下降趋势中 |
| **分手线** | 同价开盘、颜色相反的两根 | 持续/反转取决于趋势方向 |
| 向上跳空未回补 | 上升动能强 |
| 跳空快速回补 | 动能不足 |

### 粗细蜡烛图（page-0405）

- **概念**：蜡烛实体**粗细**由当日成交量决定——量越大线越粗（Morris 粗细蜡烛）。
- **读法**：上升趋势中上涨日应更粗、回调日更细；反之亦然。
- 与长实体+放量规则一致；本地 MVP 无粗细渲染，用 **`volume_ratio`** 与实体大小**近似**判读。

### 位置 × 形态

| 位置 | 锤头 vs 上吊 | 双 K 反转 | 解读 |
|------|-------------|-----------|------|
| 下跌末段+支撑 | 锤头 | 刺透/吞没/启明星 | 质量高 |
| 上升末段+阻力 | 上吊 | 乌云/吞没/黄昏星 | 质量高 |
| 横盘中部 | 任意 | 任意 | 权重低，忽略或降级 |
| 上升中段 | 锤头形似 | — | 多为中继，非反转 |
| 涨跌停/一字 | 影线失真 | — | 不硬判 |

### 第 12 章过滤链（反转形态必过）

振荡指标进入**预备区**后，才采信蜡烛**反转**形态；**持续形态不过滤**。

**「曾」= 近 20 个交易日内**（本轮 swing，非历史任意时点；详见 `ta-oscillators` 预备区专节）。

**预备区定义**（近 20 日内满足其一即可作过滤）：

| 指标 | 看多预备区（才采信看涨反转 K） | 看空预备区（才采信看跌反转 K） |
|------|-------------------------------|-------------------------------|
| KDJ 的 **D**（≈慢速随机 %D） | 近 20 日内 **曾 <20**，或当前 <20 | 近 20 日内 **曾 >80**，或当前 >80 |
| RSI(14) | 近 20 日内 **曾 <30**，或当前 <30 | 近 20 日内 **曾 >70**，或当前 >70 |
| CCI | 近 20 日内 **曾 <−100**，或当前 <−100 | 近 20 日内 **曾 >+100**，或当前 >+100 |
| Williams %R | 近 20 日内 **曾 <20**，或当前 <20 | 近 20 日内 **曾 >80**，或当前 >80 |

**操作规则**：

| 情形 | 处理 |
|------|------|
| 看涨反转 K + 近 20 日 D/RSI **未**进看多预备区（如 D=65） | **忽略**该形态 |
| 看跌反转 K + 近 20 日 D/RSI **未**进看空预备区 | **忽略** |
| 近 20 日内 D 先 <20 后再升回 >20 + 刺透/锤头等 | 买入信号形成区，质量↑ |
| 近 20 日内 D 先 >80 后再回落 + 乌云/黄昏星等 | 卖出信号形成区，质量↑ |
| 一年前 D>80、近 20 日 D 一直在 40～60 + 反转 K | **忽略**（预备区已失效） |
| 反转 K + 预备区 + 放量 | 质量最高 |
| 反转 K + 预备区 + 缩量 | 可采信但降权 |
| 持续 K 线（三法等） | 不看预备区；看趋势是否同向 |

也可用 MACD 背离、CCI、Williams %R 作同类过滤（逻辑同：先极端区，再认 K 线）。

### 与 ta-oscillators 协同

1. `ta-trend-structure` 定趋势与位置  
2. 本 Skill 识别形态  
3. 查预备区（`d`/`rsi14`）→ 未进预备区则反转形态**无效**  
4. `ta-oscillators` 查背离/钝化/交叉作二次确认  

### 常见误判

- 孤立形态不看位置（锤头/上吊同形不同义）
- 十字线单独定方向
- **反转形态不过滤预备区**（第 12 章核心）
- **把历史久远极端区当「曾」**（须近 20 日内；一年前 D>80 不算）
- 吞没/刺透/乌云无次日确认
- 涨跌停影线失真仍硬判
- 持续形态出现在逆趋势环境

### 失效条件

- 无 OHLC
- 仅 1 根 K 线无确认（除明确单根规则）
- 横盘中部权重低
- 缺 `d`/`rsi14` 时：反转结论须标注「预备区未验证」

## 分析步骤

1. `ta-trend-structure` 或峰谷定当前趋势与高低位
2. 读近 5～10 根 OHLC；`analyze_candlestick` 识别形态
3. 判反转 vs 持续
4. **反转**：查 D/RSI 是否进预备区 → 未进则忽略
5. 查位置表 + 量比
6. 输出形态 + 过滤结果 + 综合判断

## 禁止事项

- 反转形态不过滤预备区即下结论
- 孤立形态不看位置与量
- 无 OHLC 仍识别

### 汇总表其余形态（page-0287～0288）

弃婴、三川、南方三星等 **30+ 日式稀有形态**不在 MVP 逐条判据内。识别到类似结构时：按**位置 + 预备区 + 量能**类比启明星/黄昏星/孕线处理，或标注「稀有形态，人工确认」。

# 安全政策

## 报告漏洞

如发现安全问题，请通过 GitHub Security Advisory 或私信仓库维护者报告。**请勿在公开 Issue 中粘贴 API Key、`.env` 内容或数据库导出。**

## 设计边界

Profit Protector 是**本地单机工具**，默认绑定 `127.0.0.1:7860`，不提供公网部署方案。

| 风险点 | 现状 | 建议 |
|--------|------|------|
| API Key 存储 | 写入本地 `.env` | 勿提交版本库；勿分享 `.env` 文件 |
| `POST /api/settings` | 无鉴权，可写 `.env` | **仅在本机使用**；勿暴露到局域网/公网 |
| LLM 请求 | 用户自配 Key，数据发往百炼/Ollama | 了解供应商隐私政策 |
| SQLite 数据 | 持仓、聊天记录存本地 | 定期备份；开源前清空个人数据 |

## 密钥检查清单

推送代码前确认：

```powershell
git check-ignore -v .env data\portfolio.db
git status   # 不应出现 .env 或 *.db
```

## 金融合规

本工具**不提供**证券买卖、下单、代客理财功能。所有输出为技术分析视角，不构成投资建议。使用者须自行承担投资风险。

## 支持的版本

仅维护仓库 `master` / `main` 最新版本。
